SG212HRD24  24VDC Powered, 6 DC Inputs, 2 Analog Inputs, 4 Relay Outputs expandable up to 56 I/O, 

The SG2-12HR-D features a 4x16 character LCD screen and keypad for operator feedback and control

Programming can be in either Ladder Logic or Function Block. (*)
The programming software SG2 Client runs on Windows

A programming cable interface part number SG2USB is required (see below)

 Specifications:

Power: 20.4~28.8 VDC 100mA
Digital Inputs: 6 Inputs, OFF <5VDC, ON>15VDC
Digital Outputs: 4 Relay outputs 8 Amp
Output Rating: 265VAC / 30VDC Relay Outputs, 8A Resistive Load
Analog Inputs: 2 Analog  0-10VDC 10bit resolution
Expansion: Options up to 56 I/O
LCD Display: 4 line, 16 character
Keypad: 8 Programming Keys
Program Memory: 300 Lines or 260 Function Blocks
Program Storage: Retentive Flash Memory
Real Time Clock: Built In - Daylight Saving Feature
Timers: 31 (0.01Sec ~ 9999 min)
Power Drain: 150mA @12V,  90mA @24V
Approvals: UL / cUL / CE
Warranty: 1 Year
